# cybersecurity-home-lab
Cybersecurity Home Lab – Incident Response & Network Analysis

This repository contains hands-on incident response investigations using public PCAP datasets. 
The goal is to practice network traffic analysis, threat detection, and report writing.

Tools Used:
- Kali Linux
- Wireshark
- tcpdump
- Zeek (optional)
- Suricata (optional)

Skills Demonstrated:
- PCAP analysis
- Threat hunting
- IOC identification
- Network forensics
- Incident reporting
